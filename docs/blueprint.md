# **App Name**: CardWise

## Core Features:

- Card Display: Display a single question card centered on the screen.
- Action Button: Implement a primary action button below the question that changes its functionality (e.g., 'Show Answer' / 'Next').
- Theme Toggle: Add a light/dark theme toggle button fixed at the bottom-left that switches theme instantly on click.
- Admin Authentication: Secure the admin panel with authentication to control access.
- Card Management: Enable admins to create, edit, delete, and manage question-answer cards.
- Dashboard Table: Provide a dashboard table in the admin panel to view all cards and their status.
- Data Persistency: Persist card data to the MongoDB Atlas database.

## Style Guidelines:

- Primary color: Dark green (#006400) to inspire trust and focus.
- Background color: White (#FFFFFF) for light mode, Black (#000000) for dark mode.
- Accent color: White (#FFFFFF) for action buttons and highlights in dark mode, Black (#000000) in light mode.
- Body and headline font: 'Inter', a grotesque-style sans-serif font for a clean, neutral, objective look.
- Use a clean, minimal, and responsive design with centered question display.
- Simple, clear icons for the hamburger menu and theme toggle button.
- Subtle transitions for theme changes and card transitions.
